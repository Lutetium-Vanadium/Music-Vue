Updates

When a new update is released to GitHub, you will be notified on next time you open the app to update. To update, go to the directory where you cloned the files and run the update.sh file.

This will keep all the changes you've made.

Player Controls

Toggling the like button updates the database, but currently doesn't change in the controls immediately

Shortcuts

While the app is focused:

Space

Play / Pause

Cmd/Ctrl+Left

Previous Track

Cmd/Ctrl+Right

Next track

Up

Increase Volume

Down

Decrease Volume

S

Shuffle Songs

L

Loop Current Song

PageDown

Jump Back

Left

Seek Back

Right

Seek Forward

PageUp

Jump Forward

Play / Pause

Media PrevTrack

Media NextTrack

Next track

Media Stop
